SpinDance Weather Reporter


How to process a set of raw weather samples:

1. Place the .csv file to be processed into the 'csvFiles' folder.


2. Type the filename of that file (without the .csv extension).


3. Enter the Sample Size to include in each group report.

(If left blank one report will be created for the entire file.)

4. Enter the Interval to offset before the next report is generated.

(If left blank one report will be created for the entire file.)
